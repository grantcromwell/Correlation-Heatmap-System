# Correlation Heatmap Frontend

Frontend application for the Correlation Heatmap System.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
VITE_API_URL=http://localhost:8000/api/v1
VITE_WS_URL=ws://localhost:8000/ws
VITE_GRAPHQL_URL=http://localhost:8000/graphql
```

3. Start development server:
```bash
npm run dev
```

## Development

### Type Checking
```bash
npm run type-check
```

### Linting
```bash
npm run lint
```

### Formatting
```bash
npm run format
```

### Testing
```bash
npm run test:unit
```
